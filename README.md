# Inventory Manager
Inventory Manager is the backend service for an inventory management system.

Front end:
- https://github.com/Psuedoo/inventory_manager_vue

# Local Development
Requirements:
- Docker

Run:
```
docker-compose up -d
```

Visit the services:

- Adminer: http://localhost:8080
  
  Login Credentials:
    - System: PostgreSQL
    - Server: db
    - Username: admin
    - Password: dbpass
    - Database: admin
    
- FastAPI Docs: http://localhost:8000/docs
